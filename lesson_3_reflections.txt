Describe the differences between forks, clones, and branches. When would you use one instead of another?
Forks: Realiza una copia identica de un repositorio (fuente original) hacia el nuestro manteniendo el credito del autor original (solo se realiza dentro de github).
Clones: Realiza una copia identica de un repositorio desde gitHub hacia nuestra computadora.
Branches: Son ramificaciones dentro de nuestro repositorio local
